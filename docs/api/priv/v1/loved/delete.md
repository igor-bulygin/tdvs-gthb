### Loved - Delete

Example about how to call to Web Service to delete a loved product for the connected user

**URL**: `/api/priv/v1/loved/<:product_id>`

**Method**: `DELETE`

**Response codes**:
* `204`: No contect
* `400`: Bad request
* `401`: Unauthorized 
* `403`: Forbidden

**Request body**: No body
