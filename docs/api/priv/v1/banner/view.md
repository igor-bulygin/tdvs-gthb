### Banners - View (GET detail)

Example about how to call to Web Service to get specific banner

**URL**: `/api/priv/v1/banner/<:id>`

**Method**: `GET`

**Response codes**: 
* `200`: Success
* `400`: Bad request
* `401`: Unauthorized
* `404`: Not found
  
**Request parameters**:
* `:id`: Id of the banner that want to get
  
**Response body**:

See the structure of a banner in index.md
