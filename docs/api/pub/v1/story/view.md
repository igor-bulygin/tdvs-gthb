### Story - View (GET detail)

Example about how to call to Web Service to get specific story

**URL**: `/api3/pub/v1/story/<:storyId>`

**Method**: `GET`

**Response codes**: 
* `200`: Success
* `400`: Bad request
  
**Request parameters**:
* `:id`: Id of the story that want to get
  
**Response body**:

See the structure of a story in index.md
