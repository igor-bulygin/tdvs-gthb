### Loved - View (GET detail)

Example about how to call to Web Service to get specific Loved

**URL**: `/api3/pub/v1/loved/<:id>`

**Method**: `GET`

**Response codes**: 
* `200`: Success
* `400`: Bad request
  
**Request parameters**:
* `:id`: Id of the loved that want to get
