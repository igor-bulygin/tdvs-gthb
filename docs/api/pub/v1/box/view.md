### Boxes - View (GET detail)

Example about how to call to Web Service to get specific box

**URL**: `/api3/pub/v1/box/<:id>`

**Method**: `GET`

**Response codes**: 
* `200`: Success
* `400`: Bad request
  
**Request parameters**:
* `:id`: Id of the box that want to get

See index.md
