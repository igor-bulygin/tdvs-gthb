### Countries - EU Countries (GET list)

Example about how to call to Web Service to get a public list of all Countries in the European Union

**URL**: `/api3/pub/v1/countries/eu-countries`

**Method**: `GET`

**Response codes**: 
* `200`: Success
* `400`: Bad request
  
**Request parameters**:
* No params
